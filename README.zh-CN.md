**MoonImViewer** 是一款为需要快速浏览**大量图片文件夹**的用户打造的桌面图片查看器。

它不需要你一张张手动打开图片，而是提供快速的缩略图网格、专注的详细查看流程，以及帮助你保留重要图片的轻量功能。

## 开始使用

[![下载最新版本](https://img.shields.io/badge/Download-Latest%20Build-111827?style=for-the-badge&logo=github)](https://github.com/msjun23/MoonImViewer/releases/latest)

使用 MoonImViewer 最简单的方式，就是从 GitHub Releases 下载最新构建版本。

- macOS：`MoonImViewer-xxx-macos.zip`
- Windows：`MoonImViewer-xxx-windows.zip`

### 首次启动权限说明

#### macOS

1. 打开 `系统设置`。
2. 前往 `隐私与安全性`。
3. 在靠近底部的位置找到被拦截的应用提示。
4. 点击 `仍要打开`。

#### Windows

1. 右键点击已下载的应用文件。
2. 选择 `属性`。
3. 在 `常规` 选项卡中，如果出现 `解除锁定`，请勾选它。
4. 点击 `应用`，然后再次启动应用。

## 键盘快捷键

### 画廊视图

- `Arrow keys`：移动选择
- `Space`：打开当前选中的图片

### 详细查看

- `Q/q`：关闭详细查看
- `Left` / `Right`：上一张或下一张图片
- `Enter`：收藏或取消收藏图片
- `A/a` 或 `Space`：开始或停止自动幻灯片播放
- `Up` / `Down`：调整幻灯片播放速度
- `M/m`：切换放大镜
- `Mouse wheel`：缩放
- `Drag`：拖动画面
