**MoonImViewer** is a desktop image viewer built for people who move through **large folders of images** quickly.

Instead of making you open files one by one, it gives you a fast thumbnail grid, a focused detailed-view flow, and lightweight function for keeping the images that matter.

## Getting Started

[![Download Latest Build](https://img.shields.io/badge/Download-Latest%20Build-111827?style=for-the-badge&logo=github)](https://github.com/msjun23/MoonImViewer/releases/latest)

The simplest way to use MoonImViewer is to download the latest app build from GitHub Releases.

- macOS: `MoonImViewer-xxx-macos.zip`
- Windows: `MoonImViewer-xxx-windows.zip`

### First Launch Permission Notes

#### macOS

1. Open `System Settings`.
2. Go to `Privacy & Security`.
3. Find the blocked app message near the bottom.
4. Click `Open Anyway`.

#### Windows

1. Right-click the downloaded app file.
2. Choose `Properties`.
3. In the `General` tab, enable `Unblock` if it appears.
4. Click `Apply`, then launch the app again.

## Keyboard Shortcut

### Gallery

- `Arrow keys`: move selection
- `Space`: open selected image

### Detailed View

- `Q/q`: close detailed view
- `Left` / `Right`: previous or next image
- `Enter`: like or unlike image
- `A/a` or `Space`: start or stop auto slideshow
- `Up` / `Down`: change slideshow speed
- `M/m`: toggle magnifier
- `Mouse wheel`: zoom
- `Drag`: pan image
